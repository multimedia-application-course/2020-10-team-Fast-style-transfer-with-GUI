## Train
自己训练的时候要先找数据集。 推荐数据集大小要大于4W张图片。
```
python train.py   --dataset_path <path-to-dataset> \
                  --style_image <path-to-style-image> \
                  --epochs 1 \
                  --batch_size 4 \
                  --image_size 256
```
